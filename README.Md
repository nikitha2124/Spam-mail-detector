# 📧 Spam Mail Detector

A Machine Learning and NLP-based application that classifies messages as Spam or Ham using TF-IDF vectorization and classification algorithms.

## Features

* Text Preprocessing
* Stopword Removal
* TF-IDF Vectorization
* Spam/Ham Classification
* Confidence Score Prediction
* Confusion Matrix Evaluation
* Model Comparison (Naive Bayes & Logistic Regression)
* Streamlit Web Application

## Technologies Used

* Python
* Scikit-Learn
* Pandas
* NumPy
* NLTK
* Streamlit

## Project Structure

spam-mail-detector/

## Run the Project

Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run app.py
