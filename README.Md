# 🚀 AI-Powered Stock Analytics Platform

A full-stack stock market analytics platform that provides real-time stock insights, AI-powered investment recommendations, portfolio tracking, and trade history management.

## 📖 Overview

The AI-Powered Stock Analytics Platform helps users analyze stock market trends and make informed investment decisions through real-time data visualization and AI-generated recommendations.

## ✨ Features

### 📊 Real-Time Stock Dashboard

* Live stock price tracking
* Market trend analysis
* Interactive TradingView charts
* Stock search functionality
* Real-time updates using Socket.io

### 🤖 AI Analytics Engine

* Buy/Sell recommendations
* Confidence score generation
* Market trend prediction
* Risk assessment analysis

### 💼 Portfolio Management

* Track stock holdings
* Monitor investment performance
* Calculate profit/loss

### 📜 Trade History

* Record Buy/Sell transactions
* Store historical trades in MongoDB
* View transaction history

## 🛠️ Tech Stack

### Frontend

* React.js
* JavaScript
* HTML5
* CSS3

### Backend

* Node.js
* Express.js
* Socket.io

### Database

* MongoDB
* Mongoose

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/stock-ai-analyzer.git
cd stock-ai-analyzer
```

### Backend Setup

```bash
cd server
npm install
npm start
```

### Frontend Setup

```bash
cd client
npm install
npm run dev
```

## 🔑 Environment Variables

```env
MONGO_URI=your_mongodb_connection_string
ALPHA_VANTAGE_API_KEY=your_api_key
PORT=5000
```
